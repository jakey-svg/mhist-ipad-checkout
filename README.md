# MHIST iPad Checkout

Browser-based iPad check-in and check-out app for Mill Hill International School Thailand.

## Suggested GitHub repository name

`mhist-ipad-checkout`

## GitHub Pages

Because the app is a single static `index.html` site, it can be published directly with GitHub Pages.

## Notes

- Data is currently stored in browser local storage.
- That means each browser keeps its own records unless a shared backend is added later.
